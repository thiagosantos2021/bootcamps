Post #1 - Definição HTML5

HTML5, sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web. Ela traz novas funções como as tags <video>, <audio> e elemento <canvas> e será o novo padrão para HTML, XHTML e HTML DOM. Com o HTML5, por exemplo, é possível ver vídeos do YouTube sem o Flash.
